# REPTILE PROJECT

## Get Started
### Clone the repo
```bash
git clone git@github.com:bgreen55/Reptile-Tracker.git
```
then change directories into it

```bash
cd Reptile-Tracker
```

### Install the dependencies

With yarn
```bash
yarn
```

With npm
```bash
npm install
```

## Development
### Running the server
Start the server by running:

With yarn
```bash
yarn dev
```

With npm
```bash
npm run dev
```

## Production
Build the project by running

With yarn
```bash
yarn build
```

With npm
```bash
npm run build
```
